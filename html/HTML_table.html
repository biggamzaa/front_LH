  1. 테이블 형식 요소
    • tr : 테이블 제작시 필수 이며, 하나의 행(row)를 생성한다.
    • td : 일반적으로 내용을 포함할 때 사용한다.
    • th : 테이블에서 제목이 있는 테이블에 사용한다.
  2. 테이블 내용 요소
    • thead : 테이블에서 제목이 들어가는 부분을 묶을 떄 사용한다.
    • tbody : 내용이 들어가는 테이블을 묶을 때 사용한다.
    • tfoot : 테이블 마지막 부분을 묶을 떄 사용한다.
  3. 테이블 칸 합치는 방법
    • colspan : 열(column)을 병합할 때 사용한다.
    • rowspan : 행(row)을 병합할 때 사용한다.
