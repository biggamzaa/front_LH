HTML 기초

  1. HTML 개요
    • HTML이란 웹 페이지를 구성하는 마크업 언어이다.
  2. 단락 요소
    • p 태그로 단락을 생성할 수 있다.
    • ex) <p>단락을 생성하는 태그</p>
  3. 제목 요소
    • hn으로 제목, 소제목을 생성할 수 있다.

    • h1은 페이지에서 하나만 존재할 수 있다.

    • h2는 여기 하나

    • h2는 여기 둘, 이렇게 여러개 생성 가능하다.

    • 얘는 h3 세번째 제목이다.

  4. 목록 요소
    • ul 태그로 번호를 붙이지 않고 생성하는 목록 태그
    • ol 태그로 번호를 붙여서 생성하는 목록 태그
    • 지금 이 정리글은 ol과 ul을 섞어서 제작하는 중이다.
    • 목록을 생성하고 태그 안에 li를 포함하여 마크업할 수 있다.
  5. 앵커 태그
    • a로 다른 페이지로 이동할 수 있는 문장을 생성할 수 있다.
    • 이동하고자하는 페이지를 a 태그 안에 href를 첨부해야 한다.
  6. 이미지
    • 이미지를 추가할때는 img 태그안에 src를 첨부해서 이미지를 추가할 수 있다.
    • 첫 페이지에 있는 html 이미지도 같은 방법으로 추가하였다.
    • alt로 이미지의 설명(?)을 추가가능하다.
    • width로 이미지의 크기를 정할 수 있다.(놀라운 사실 width를 한글로치면 "장소"이다.)
  7. 주석
    • 단축키는 crtl+/이다.
    • (여기 이스터에그 있어여)
  8. 특수문자 출력
    • HTML에서 특수문자를 출력 할때는 Entity code를 사용해서 출력할 수 있다.
    • 코드 &와 ;를 포함하고 있기 떄문에 구분하기 쉽다.
