HTML 폼

  1. form : 데이터를 입력받을 수 있는 입력 폼들을 묶을 때 사용한다.
    • action : form 태그 내부에 있는 데이터를 서버로 보낼 때, 데이터가 도착할 URL을 포함한다.
  2. label : 폼 내부의 데이터를 입력받는 태그의 설명을 기입할때 사용한다.
    • for : input으로 받아들이는 데이터의 id와 연결되서 label을 클릭해도 input이 작동한다.
  3. input : 폼에서 어떠한 데이터를 입력받을 때 사용한다.
    • type : input으로 어떤 요소를 입력 받을 것인지 선택할 수 있다. 종류는 매우 다양하다.
    • name : 폼 내부에서 input으로 입력받은 데이터를 참조하여 action에 사용할 이름을 포함한다.
    • id : 해당 태그의 고유 식별을 목적으로 기입하여 사용한다.
    • button : 폼 내부에서 상호작용하는 버튼을 생성한다.
      • type : input보다는 적지만 어떤 입력을 수행하는 버튼인지 선택할 수 있다
